# synoctl brand assets

Minimal brand assets derived from the terminal mark used in the TUI.

## Files

- `synoctl-mark.svg` - square app/repo mark, suitable for avatars, favicons, and small placements.
- `synoctl-lockup.svg` - transparent terminal-style logo lockup.
- `synoctl-wordmark.svg` - horizontal lockup for docs and README use.
- `github-social-preview.svg` - source for GitHub's repository social preview.
- `github-social-preview.png` - raster export for GitHub upload.

GitHub repository social previews should be PNG, JPG, or GIF, under 1 MB. GitHub recommends at least 640 by 320 pixels, with 1280 by 640 pixels preferred.
